# Task Management System

A modern and responsive Task Management System built using React and Vite. This application helps users organize and track tasks efficiently using a Kanban-style board with Todo, In Progress, and Done sections.

## Features

* Add new tasks
* Move tasks between Todo, In Progress, and Done
* Task count for each section
* Responsive user interface
* Local Storage support (tasks remain after page refresh)
* Modern dashboard design
* Easy-to-use task management workflow

## Technologies Used

* React.js
* Vite
* JavaScript (ES6+)
* HTML5
* CSS3

## Installation

1. Clone the repository

```bash
git clone https://github.com/keerthi-523/Task-Management-System.git
```

2. Navigate to the project folder

```bash
cd Task-Management-System
```

3. Install dependencies

```bash
npm install
```

4. Start the development server

```bash
npm run dev
```
